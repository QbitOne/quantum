How-To-Use

Der Styleguide soll eine Übersicht liefern, wie das CSS des Quantum Themes aufgebaut ist.

Zusätzlich soll er als Unterstützung im Development dienen, da alle genutzen CSS-Styles aufgeführt werden.

Atomic Design

Das Quantum Theme nutzen den Atomic Design Ansatz.
_Siehe Brad Frost - Atomic Design

Hierbei wird von den kleinstmöglichen Elemente ausgehend, ein System aufgebaut.

  • atoms

    • buttons, paragraphs, headings

  • molecules

    • input and label, searchbar and button

  • organisms

    • header or footer

Zudem werden im Quantum unter generic und nucleus weitere Styles beschrieben.

Generic

Global Variables

Quantum nutzt globale CSS Variablen, welche widerum im Child Theme überschrieben werden können.

:root {
  --max-site-width: 1440px;
  --site-spacing-left: 5%;
  --site-spacing-right: 5%;
}

Breakpoints

Quantum nutzt 4 vordefinierte Breakpoints, welche als Mixin innerhalb PostCSS eingesetzt werden können.
Inspiriert von TailwindCSS Breakpoints

Zu beachten:
Diese Breakpoints sind nur für das Quantum Theme nutzbar.
Da sie als mixin innerhalb postcss verwendet werden, sind sie nicht für das Projekttheme anwendbar.
Es muss innerhalb des Projekttheme ebenfalls die _breakpoints.css angelegt werden.

@define-mixin atSmall {
  @media (min-width: 640px) {
    @mixin-content;
  }
}

@define-mixin atMedium {
  @media (min-width: 768px) {
    @mixin-content;
  }
}

@define-mixin atLarge {
  @media (min-width: 1024px) {
    @mixin-content;
  }
}

@define-mixin atExtraLarge {
  @media (min-width: 1280px) {
    @mixin-content;
  }
}

Farben

Quantum nutzt Grautöne als Standardfarben. Das ist eine zweite Zeile.

@color: hsl(0, 0%, 50%) @name: Primary
@color: hsl(0, 0%, 75%) @name: Secondary
@color: hsl(0, 0%, 75%) @name: Accent
@color: hsl(0, 0%, 20%) @name: Text

Typography

Ausgehend von einer Basis Schriftgröße der Browser von 16px nutzt Quantum eine einfache Umrechnung von px in die Einheit rem.

1rem = 10px

Als Fallback Schriftart wird standardmäßig sans-serif genutzt.

html {
    font-size: 62.5%;
    font-family: sans-serif;
}

Die Standard Schriftgröße des Quantum Themes ist auf 18px festgelegt.

body {
    font-size: 1.8rem;
    line-height: 1.5;
}
