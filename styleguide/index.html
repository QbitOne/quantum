Breakpoints

Es sind zwei Breakpoints definiert.

  • 800px

  • 1200px

Zu beachten:
Diese Breakpoints sind momentan nur für das Quantum Theme nutzbar.
Da sie als mixin innerhalb postcss verwendet werden, sind sie nicht für das Projekttheme anwendbar.
Dafür muss momentan innerhalb des Projekttheme ebenfalls die _breakpoints.css angelegt werden.

@define-mixin atSmall {
    @media (min-width: 800px) {
        @mixin-content;
    }
}

@define-mixin atMedium {
    @media (min-width: 1200px) {
        @mixin-content;
    }
}

Farben

Quantum nutzt Grautöne als Standardfarben. Das ist eine zweite Zeile.

@color: hsl(0, 0%, 50%) @name: Primary
@color: hsl(0, 0%, 75%) @name: Secondary
@color: hsl(0, 0%, 75%) @name: Accent
@color: hsl(0, 0%, 20%) @name: Text

Typography

Ausgehend von einer Basis Schriftgröße der Browser von 16px nutzt Quantum eine einfache Umrechnung von px in die Einheit rem.

1rem = 10px

Als Fallback Schriftart wird standardmäßig sans-serif genutzt.

html {
    font-size: 62.5%;
    font-family: sans-serif;
}

Die Standard Schriftgröße des Quantum Themes ist auf 18px festgelegt.

body {
    font-size: 1.8rem;
    line-height: 1.5;
}
